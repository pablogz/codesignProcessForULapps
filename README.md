# Proceso de codiseño
Proceso de codiseño que permite recoger las características de los dominios educativos para facilitar el diseño y desarrollo de aplicaciones de aprendizaje ubicuo.

Este proceso ha sido definido en el marco de la tesis doctoral de [Pablo García Zarza](https://www.linkedin.com/in/pabgarzar) dentro del grupo de investigación [GSIC-EMIC](https://gsic.uva.es).
